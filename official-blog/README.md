# official-blog/

PRONI **公式ブログ（会社アカウント）** 向け記事の下書き置き場。

- 個人の Zenn / Qiita / note 記事（`articles/` `Qiita/` `articles_note/`）とは**別トラック**。
- 公開先は会社の note（`note.proni.co.jp`）等で、**公開は手動**（このリポジトリの publish パイプラインの対象外）。
- `npm run check` 等の個人媒体向け構造チェックの対象には含めない。
- 社内固有情報（内部スキル名・内部リポジトリ名・KPI 具体数値・体制等）は**持ち込まない**（サニタイズ済みのみ）。
